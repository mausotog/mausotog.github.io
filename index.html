Research

    My current research encompasses primarily problems relating machine learning applied to time series sensor data. I also publish and review a lot in software engineering conferences and journals. Some of my topics of expertise include automated repair, fault detection, software quality, testing, etc. During my PhD I worked with my advisor Claire Le Goues creating automatic program repair approaches which combine stochastic search methods like genetic programming with lightweight program analyses to find patches for real bugs in existing software. In my latest studies we propose a set of mechanisms to increase the quality of plausible patches in automatic program repair, including analyzing developer behavior to inform the mutation operator selection distribution, an analysis of test suite behavior and their key characteristics for automatic program repair, and a study of patch diversity as a means to create consolidated higher quality fixes.
